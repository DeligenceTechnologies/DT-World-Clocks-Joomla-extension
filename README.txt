DT WORLD CLOCKS (mod_dtworldclocks) is a clean and simple but very functional clock module for Joomla 3.X.It is adjusted for Daylight Saving Time rules. You can show upto 4 clocks.
 
The settings for the module are the following:

* Time format, 24h or 12h.
* Lets you choose if the displayed timezone and city/country name should be displayed or not.
* Whether or not seconds should be displayed.
* Whether or not the current date should be displayed and if yes, then in what format.
* Multiple Date Formats
* whether to display leading zeros in Hours.
